# CC Switch for HarmonyOS

Claude Code、Codex、Gemini CLI 的全方位管理工具 — 鸿蒙 PC 版

基于 [CC Switch](https://github.com/farion1231/cc-switch) 移植到 HarmonyOS NEXT PC。

## 功能

- **提供商管理** — 管理 Claude Code、Codex、Gemini 等 API 配置，一键切换
- **MCP 服务管理** — 统一管理 Model Context Protocol 服务器
- **提示词管理** — 管理各应用的系统提示词
- **技能管理** — 安装和管理 AI 技能插件
- **多语言** — 中文/英文界面
- **深色/浅色主题**

## 开发环境

- DevEco Studio 6.0+
- HarmonyOS NEXT PC SDK

## 构建

1. DevEco Studio 打开项目
2. 连接鸿蒙 PC 设备或启动模拟器
3. 点击 Run
